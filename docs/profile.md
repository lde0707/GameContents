## 자기소개

> 객관적인 판단이 뛰어나고, 하나를 시키면 두 개를 할 줄 아는 성실한 사람

### 성격 키워드
**씩씩한 / 현명한 / 착함**

### 강점
- 긍정적인 성격
- 높은 이해능력
- 상대방을 이해시키는 전달능력

### 약점
- 암기가 약한 편 (논리로 보완)

### 성격 특성
- **완벽주의 성향** — 하나를 하면 쉽게 다음으로 넘기지 않음
- **논리적 문제 접근** — 직관보다 논리로 분석
- **리더형** — 팔로워인 줄 알았는데 책임지기 싫어하는 리더형
- **스트레스 내성 강함** — 크게 받지 않는 편
- **피드백 수용** — 겸허히 받아들임
- **빠른 환경 적응** — 새로운 환경에 잘 적응
- **객관적 갈등 중재** — 감정보다 객관적 판단 우선

### 주변 평가
착하다 / 열심히 한다 / 공부 잘한다
